# Contributing to bankadsapi

Thanks for contributing.

## How to contribute

1. Fork the repository and create a feature branch.
2. Make focused changes with clear commit messages.
3. Run local checks and ensure the project starts successfully.
4. Open a pull request with:
   - a clear description of what changed
   - why the change is needed
   - any testing details

## Contribution requirements

- By submitting a pull request, you agree to the terms in `CLA.md`.
- Do not submit code you do not have rights to contribute.
- Keep changes scoped to one problem whenever possible.

## Code style

- Follow existing project conventions.
- Prefer small, readable functions.
- Add or update docs when behavior changes.
